# discord_bot ✉️

## 디스코드 봇을 만드는 개인프로젝트 입니다!

> 디스코드 봇의 기능을 하나씩 만들어보며 명령어를 통해 안내하는 봇을 만드는 프로젝트 입니다

> 현재 기능은 인사 | 프로필 사진 추출 | 이모지 보여주기 | gif 보여주기 | 강퇴하기 | 개인 프로필 보여주기 가 있습니다
